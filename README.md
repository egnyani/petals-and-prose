# Petals & Prose

A web app for building custom flower bouquets and sending handwritten letters — built with React.

## What it does

- **Arrange** — Pick from 14 hand-illustrated SVG flowers, drag them around the canvas to build your bouquet, and delete any you don't want
- **Write** — Compose a letter with 6 handwriting fonts, 6 paper styles, custom font size, and a subject line
- **Send** — Address it to someone with a name and email, preview the full bouquet + letter, and copy it to send

## Features

- 14 unique SVG flower illustrations (rose, tulip, sunflower, daisy, blossom, poppy, orchid, hibiscus, lotus, lily, pansy, wildflower, rosette, clover)
- Drag-to-arrange with organic curved stems
- Hover any flower to reveal a delete button
- Decorative bouquet paper wrap
- Undo / Clear history
- Character counter (600 max)
- Email validation
- Toast notifications
- Fully responsive

## Stack

- React 18 (hooks only, no external state library)
- SVG illustrations (no emoji, no icon libraries)
- Google Fonts — Syne + Space Grotesk

## Run locally

Just open `preview.html` in any browser — no build step needed.

For the React component, drop `flower-letter.jsx` into any React project.

## Color palette

Inspired by bold, saturated earth tones — terracotta, dusty pink, forest green, sage, peach, chartreuse, periwinkle, and dark teal.
